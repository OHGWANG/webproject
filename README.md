## 프로젝트 정보 

### 1.프로젝트 주제

서울 음식점 추천 사이트

### 2.주제 선정배경 또는 이유

점심시간 때  음식점 고르는 시간을 해소하기 위해 음식점 추천 서비스를 제공하고자 합니다.
